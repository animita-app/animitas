# 🎬 Quick Recording Reference

## 🚀 Quick Start (3 Steps)

1. **Start dev server**: `npm run dev`
2. **Open demo**: http://localhost:3000/demo
3. **Record with Screen Studio**: 1080x1350 @ 4K

---

## 📍 Demo URLs

```bash
# Map
http://localhost:3000/demo/map

# Add Form
http://localhost:3000/demo/add

# Detail Page
http://localhost:3000/demo/detail/hermogenes

# Research Mode
http://localhost:3000/demo/research
```

- 1080 x 1350 px

## 💡 Pro Tips

1. Hide REC indicator before final recording
2. Smooth, slow mouse movements
3. Pause on key interactions
4. Add text overlays in post
5. Test recording first

## 🎬 Suggested Shots

### Map (10s)
1. Cluster hover → click
2. Individual marker → popup
3. Pan & zoom

### Form (10s)
1. Scroll through fields
2. Type in input (show autocomplete)
3. Click location picker
4. Hover upload button

### Detail (8s)
1. Hero image
2. Scroll story
3. Show insights
4. Click feedback

### Research (12s)

2. Charts appear
3. Click filter
4. Map updates
5. Show heatmap
